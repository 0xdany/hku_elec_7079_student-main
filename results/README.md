# Results Directory

This directory contains all output files generated during the project execution.

## Directory Structure

- **figures/**: Generated charts and visualizations
- **models/**: Trained machine learning models
- **reports/**: Generated analysis reports
- **logs/**: Execution logs and debugging information

## File Naming Convention

- Figures: `{task_name}_{description}_{timestamp}.png`
- Models: `{model_type}_{task_name}_{timestamp}.pkl`
- Reports: `{report_type}_{task_name}_{timestamp}.html`
- Logs: `{task_name}_{timestamp}.log`
